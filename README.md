# 自動倉庫 スタッカークレーン 3D シミュレーター

Three.js + Cannon-es によるブラウザ動作の自動倉庫物理シミュレーション。
スタッカークレーンの走行・昇降・フォーク動作を、実機に近い制御シーケンスと
インバータモータ物理モデルで再現する。

## デモ

```
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

---

## 構成

```
src/
  main.js            Three.js シーン・カメラ・ライト・メインループ
  Warehouse.js       ラック構造 (20間口 × 10段 × 両面) + ステーション ST1/ST2
  StackerCrane.js    スタッカークレーン 3Dメッシュ + 3軸モータ制御
  InverterMotor.js   インバータ駆動 AC モータ物理モデル
  AutoController.js  ジョブ実行シーケンサ (10ステップ制御)
  JobScheduler.js    ジョブキュー管理
  LoadManager.js     パレット生成・管理
  PhysicsWorld.js    Cannon-es 物理エンジンラッパー
  Motor.js           (旧) 台形プロファイルモータ (参照用)
```

## 機能一覧

### ラック構成
| 項目 | 値 |
|---|---|
| 間口数 | 20 |
| 段数 | 10 |
| 収納セル総数 | 400 (両面合計) |
| 間口幅 | 1.4 m |
| 段高さ | 1.2 m |

### ステーション
| ID | 用途 | 位置 |
|---|---|---|
| ST1 | 入庫 (デフォルト) | 通路左端・黄色プラットフォーム |
| ST2 | 出庫 (デフォルト) | 通路右端・緑色プラットフォーム |

### 棚アドレス指定
```
L-3-2  →  左ラック, 間口 3, 2段目
R-10-5 →  右ラック, 間口 10, 5段目
```

### ジョブ種別
| 種別 | 動作 |
|---|---|
| 入庫 | ST1 のパレットを指定セルへ格納 |
| 出庫 | 指定セルのパレットを ST2 へ搬出 |
| 移動 | セル A のパレットをセル B へ移載 |

## モータ物理モデル

`InverterMotor.js` が 3 軸それぞれを VFD 駆動 AC モータとしてシミュレート。

| モータ | 定格 | ギア比 | 最高速 |
|---|---|---|---|
| 走行モータ | 11 kW / 1480 rpm | 8 : 1 (車輪径 φ300) | 3.0 m/s |
| 昇降モータ | 7.5 kW / 1480 rpm | 10 : 1 (ドラム径 φ200) | 1.5 m/s |
| フォークモータ | 1.5 kW / 1480 rpm | 9 : 1 | 0.9 m/s |

- 速度制御: 台形プロファイル (加減速度を個別設定)
- テレメトリ逆算: `F = Ma + kFv + F_load` からトルク・電流・周波数を算出
- 昇降モータは荷物搭載時に重力荷重 (≈ 3730 N) を加算 → 電流・トルクが増加

### モータ監視パネル (画面右下)
リアルタイムで以下を表示:
- 回転数 [rpm]
- トルク率 [%] (100% 超過時に赤表示)
- 電流 [A]
- VFD 出力周波数 [Hz]

## ピック＆プレース シーケンス (実機準拠)

荷物の取り出し:
```
1. 下定位置へ移動  (棚レベル - 8 cm)
2. フォーク伸長    (爪をパレット底面の下に差し込む)
3. 荷物アタッチ
4. 上定位置へ昇降  (棚レベル + 8 cm)  ← パレットが棚梁から浮く
5. フォーク収納
```

荷物の格納:
```
1. 上定位置へ移動  (棚レベル + 8 cm)
2. フォーク伸長    (パレットを棚梁の上にかざす)
3. 下定位置へ降ろす (棚レベル - 8 cm) ← パレットが棚梁に着座
4. 荷物デタッチ
5. フォーク収納
```
